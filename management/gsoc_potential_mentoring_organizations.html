Potential Mentoring OrganizationsÂ¶

This is a list of organizations that PyDy students may be able to work with and apply to for a GSoC position:

  • SymPy SymPy Mechanics is the core of the PyDy workflow and software. We will continue to work with them for the symbolic side of our system.
  • Portland State University Portland State hosts various academic related projects. They funded the first year of PyDy’s development.
  • Python Software Foundation PSF is an umbrella org that can host varous projects in the Python ecosystem.
  • The Open Motion Planning Library These guys do robotics. PyDy could be super useful for them to have correctly derived dynamics.
  • Blender Foundation Blender can be linked with a physics engine to driving animations of complex scenes.
  • BRL-CAD A physics backend can add dynamic simulation capabilities BRL-CAD. Many CAD packages offer this.
  • Crystal Space Crystal space is a 3D engine with some ragdoll physics capabilities. PyDy could utilize their infastructure to give precision dynamic simulations.
  • MoveIt! Moveit needs motion planning for there system. PyDy can provide precise kinematics for them.
  • OGRE OGRE is a 3D engine. PyDy could utilize OGRE to visualize simulations.
  • Open Source Robotics Foundation The OSRF utilizes Gazebo which is based on Bullit. This software could be enhance by PyDy’s more capable and advanced simulation capabilities.
  • Processing PyDy could utilize processing for visualization.
  • RoboComp Robotics framework that could potentially utilize PyDy for maniplutor dynamics and control.
  • Sage Sage is a general purpose tool for scientific computing. The Sage system and notebook could be used as a platform for PyDy because most of the elements for PyDy’s vision are already in Sage.
